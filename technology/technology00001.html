    博客常用Git命令

    1.  git init //初始化本地git环境

    2.  git clone XXX//克隆一份代码到本地仓库

    3.  git add . 会递归地添加当前工作目录中的所有文件到缓存区.

    4.  git commit -m "这里面写你的注释" 把文件提交到本地仓库

    5.  git push -u origin master 同步本地库代码到远程库
